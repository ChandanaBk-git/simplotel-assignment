Simplotel Frontend Assignment

This project contains the solution for the Frontend Web Developer (HTML, CSS) assignment provided by Simplotel Technologies Pvt. Ltd.

The webpage is developed using only HTML and CSS without using any external libraries or frameworks such as Bootstrap or Material UI.

The layout closely follows the given design and uses a similar color scheme. The webpage is fully responsive and works properly on desktop, tablet, and mobile devices. It supports window resizing and browser zoom without breaking the layout.

Images used on the page do not stretch or pixelate, and text content does not overflow on different screen sizes.

The navigation menu adapts to mobile view using a hamburger menu. The hero section stacks correctly on smaller screens with text appearing above the image.

Project Structure:
index.html
style.css
images folder containing all image assets

The project is hosted using GitHub Pages.
